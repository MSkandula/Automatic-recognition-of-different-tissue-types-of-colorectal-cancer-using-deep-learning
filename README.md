Automatic recognition of different tissue types of colorectal cancer using deep learning

GITAM University, Hyderabad(Final semester
Designed and implemented a complete deep-learning pipeline to automatically classify eight different colorectal tissue types using the Colorectal Histology dataset (TensorFlow Datasets). The project involved end-to-end development of both a custom CNN model and a Transfer Learning model using VGG16, achieving high accuracy on medical histopathology images.
Key Contributions
Built a full image-processing workflow: dataset loading, preprocessing, normalization, and extensive data augmentation using ImageDataGenerator.
Developed a multi-layer custom CNN with Batch Normalization, Dropout, and Adam optimization to classify histopathology images.
Implemented Transfer Learning using VGG16 (ImageNet) with frozen and unfrozen layers to improve generalization.
Conducted rigorous model evaluation using accuracy curves, loss curves, confusion matrices, and classification reports.
Compared performance between the custom CNN and VGG16-based model to analyze the impact of transfer learning in medical imaging tasks.
Visualized predictions vs. true labels and plotted training/validation metrics across multiple epochs.
Technologies Used:
TensorFlow, Keras, TensorFlow Datasets (TFDS), VGG16, NumPy, Matplotlib, Seaborn, Scikit-Learn, ImageDataGenerator, Python
